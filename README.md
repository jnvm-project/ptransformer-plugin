# P-Transformer
A maven plugin for generating volatile proxies for persistent classes: classes with **@Persistent** or with **package.Annotation**.
## Setup
`mvn clean && mvn install`

## Inputs to P-Transformer
- Off-Heap library. (J-NVM)
